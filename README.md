# Character Counter

Count every character of a `string` and categories it as `vowel`, `consonants`, and `digits` using `C++` programming language.

## Usage

Navigate to `src` folder

```bash
cd src
```

Build

```bash
g++ main.cpp
```

Run `.exe` file in `src` folder
